Towards Real-Time Interactive Fluid Rendering

Daniel Brown1, Alex Coppens2,
1University of Maryland, 2University of Maryland
Paper Code

Abstract

Rendering interactive fluid in real-time has a number of applications in video-games, engineering visualization tools and animation. For many of these applications, approaches which trade physical accuracy for performance are acceptable so long as visual plausibility is retained. Existing approaches struggle to achieve real-time performance due to poor scaling and computational complexity. Our work investigates multiple possible approaches to real-time interactive fluid rendering and evaluates their feasibility and drawbacks. We describe an attempt to approximate fluid using a highly simplified particle based simulator. We outline two proposed neural network architectures for approximating fluid flow and evaluate their feasibility. We describe modifications made to an open-source SPH simulation tool meant to facilitate the generation of training data for these neural networks.
